## 🎓 Course Project Documentation Tool
**URL:** [supernoobcraft.github.io/hku-toolkit](https://supernoobcraft.github.io/hku-toolkit)

Browser-based tool for HKU students to document, organize, and export coursework projects. Current feature set:
- Multi-project collection with drag-and-drop reorder and quick move controls
- Required + optional project fields for both fast entry and detailed reflection
- Auto-infer HKU faculty/department from course codes (editable)
- Local autosave, recovery, and version history snapshots
- Evidence image upload with local persistence and per-image captions
- Preset save/apply workflow for recurring project structures
- ZIP bundle export/import for full backup and restore (including evidence)
- PDF export with optional one-project-per-page mode
- Optional skills summary section in export output
- Dark/light mode and mobile-responsive layout
