## 🎓 Course Project Documentation Tool
**URL:** [supernoobcraft.github.io/hku-project-tool](https://supernoobcraft.github.io/hku-project-tool)

Browser-based tool for HKU students to document, organize, and export coursework projects. Features:
- Create and manage course projects with detailed narratives
- Auto-infer faculty/department from course codes
- Live preview of project summaries
- Export to TXT (backup) and PDF (submission-ready)
- Drag-to-reorder and multiple sort modes
- Session-based autosave (browser storage)
- Dark/light theme support
