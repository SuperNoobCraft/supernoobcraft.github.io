      Quick Start Guide

      1. Click New Project to open the editor.
      2. Fill required fields and click Save or Update Project.
      3. Use the 3-dot menu to Edit, Duplicate, Delete, or move project up and down with keyboard-friendly controls.
      4. Drag cards for manual ordering, or choose a sort mode from the Sort dropdown.
      5. Use Export Bundle (ZIP) for full backup (including images) and Export Portfolio (PDF) for submission output.
