Help

This tool helps you build a multi-project HKU portfolio, reorder projects, and export to ZIP or PDF.

Quick Start

  1. Fill optional Portfolio Header details (name and contacts) in Project Collection.
  2. Choose a Header Theme beside the personal display switch (optional).
  3. Click New Project.
  4. Complete required fields: course code, course name, academic year, term, project type, project title, tools.
  5. Choose an Item Theme beside project type (optional).
  6. Click Save or Update Project to add the card.
  7. Repeat for additional courses.
  8. Drag cards to reorder output.
  9. Use Export Bundle (ZIP) for backup/import and Export Portfolio (PDF) for final submission.
  10. Use project 3-dot menus to save presets and editor 3-dot menu to apply them.

Tutorial With Sample Inputs

Try this mini run to verify your flow:

  1. Create project 1: Course code COMP2113, year 2025-26, term Sem 1, type Group Project.
  2. Create project 2: Course code BSIM3021, year 2025-26, term Sem 2, type Individual Assignment.
  3. Open the 3-dot menu on project 2 and click Edit, then update title.
  4. Drag project 2 above project 1.
  5. Toggle Each project on individual page on and off, then export PDF both ways.
  6. Export ZIP, refresh browser, then use Import Bundle (ZIP) to restore full data including images.

Tips

  • Use Dark mode button in the top-right navigation to switch themes.
  • Site dark mode changes the app UI only; preview/export styling follows header and item themes with a light-default baseline.
  • Role and team size appear only when Project type is Group Project.
  • Contribution and narrative text areas are optional, so quick summaries are allowed.
  • Evidence images are stored locally and restored automatically in the same browser profile.
  • Session auto-recovery restores your previous state after accidental refresh.
  • Use the top-right History button to browse and restore periodic autosave snapshots.

Troubleshooting

  • If the 3-dot menu seems unresponsive, click directly on ... first, then choose Edit or Delete.
  • If ZIP import fails, ensure the file came from this tool's Export Bundle (ZIP).
  • If PDF text looks odd, refresh preview once and export again; for best results use Chrome or Edge Save as PDF.
