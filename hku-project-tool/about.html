About

The HKU Course Project Documentation Tool is a browser-based app for HKU students to build structured, multi-course project documentation for portfolio, internship, and reflection use.

It runs fully client-side with no login and no backend. Data stays in the browser, while exports provide portable records for reuse and submission.

Current Features

  • Multi-project collection with drag-to-reorder cards.
  • Edit or delete per project using the 3-dot menu.
  • Academic year and term selection for cleaner input flow.
  • Conditional team fields for group assignments only.
  • Optional profile header (name and contacts) shown in preview and exports.
  • TXT export and TXT import for data portability.
  • PDF export with optional one-project-per-page toggle.
  • Dark mode and automatic recovery after accidental refresh.

Usability and Accessibility

  • Keyboard-friendly navigation with skip link and focus outlines.
  • Status announcements through a live region for user feedback.
  • Responsive layout for desktop and mobile screens.
  • Print-focused export styling that removes interface noise in PDF output.

Design Scope

The scope is intentionally lightweight and maintainable for coursework: clear structure, practical export outputs, and HKU-oriented workflow decisions without external services.
