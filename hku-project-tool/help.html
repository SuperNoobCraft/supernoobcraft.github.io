Help

This tool helps you build a multi-project HKU portfolio, reorder projects, and export to TXT or PDF.

Quick Start

  1. Fill optional Portfolio Header details (name and contacts) in Project Collection.
  2. Click New Project.
  3. Complete required fields: course code, course name, academic year, term, project type, project title, tools.
  4. Click Save or Update Project to add the card.
  5. Repeat for additional courses.
  6. Drag cards to reorder output.
  7. Use Export Projects (TXT) for reusable data and Export Portfolio (PDF) for final submission.

Tutorial With Sample Inputs

Try this mini run to verify your flow:

  1. Create project 1: Course code COMP2113, year 2025-26, term Sem 1, type Group Project.
  2. Create project 2: Course code BSIM3021, year 2025-26, term Sem 2, type Individual Assignment.
  3. Open the 3-dot menu on project 2 and click Edit, then update title.
  4. Drag project 2 above project 1.
  5. Toggle Each project on individual page on and off, then export PDF both ways.
  6. Export TXT, refresh browser, then use Import Projects (TXT) to restore.

Tips

  • Use Dark mode button in the top-right navigation to switch themes.
  • Role and team size appear only when Project type is Group Project.
  • Contribution and narrative text areas are optional, so quick summaries are allowed.
  • Session auto-recovery restores your previous state after accidental refresh.

Troubleshooting

  • If the 3-dot menu seems unresponsive, click directly on ... first, then choose Edit or Delete.
  • If import fails, ensure the TXT came from this tool's Export Projects (TXT).
  • If PDF text looks odd, use browser Print and choose Save as PDF (Chrome/Edge recommended).
