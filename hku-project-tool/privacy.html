Privacy

No account is required. This tool does not upload your project content to a server.

What Is Stored Locally

  • Project collection data and ordering.
  • Current draft state for accidental refresh recovery.
  • Optional profile header details (name and contacts).
  • Theme preference (light or dark mode).

What Is Not Uploaded

  • Form content is not sent to external APIs by this tool.
  • Exported TXT and PDF files are generated in-browser and downloaded directly.

File Upload Note

Evidence files (images) are handled in-browser only. Browsers do not allow restoring file input binaries after refresh, so file names may remain in data while actual files must be reselected.

Clearing Data

To remove local data, clear browser site storage for this page or use a private/incognito session when needed.
